Mix Slideshow


Description

This browser add-on allows mixing images from different sources in a single slideshow. Currently, supported sources are: Pinterest boards and local folders (via the helper application). The addon is experimental and is not listed in the official browser extension stores. Sometimes its use may trigger local rate limiting on Pinterest (a kind of temporary ban). Please report bugs here.
