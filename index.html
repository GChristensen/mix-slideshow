Mix Slideshow

Firefox Add-On


Description

This browser add-on allows mixing images from different sources in a single slideshow. Currently, supported sources are:

  • Pinterest boards.
  • Local folders (via the helper application).

For example, it is possible to create a Pinterest slideshow with images from several boards. It is also possible to add contents of local folders to the slideshow. Specify the local folders in the add-on settings (available at the about:addons page in Firefox).

The add-on is experimental. Please report bugs here.
